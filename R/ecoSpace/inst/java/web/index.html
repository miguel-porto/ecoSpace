Highlights

  • Uncover the hidden biogeographic/ecological structure in species assemblages
  • Upload species lists and explore the ecological network they create based on user-selected bioclimatic variables
  • Change properties of the network structure and change the underlying distance matrix on the fly, immediately observing the resulting structural network changes
  • Integrate network information on external apps by using the API webservices
  • Embed an ecoSpace widget in your webpage
  • Download the analysis results, the underlying distance matrix and the network itself as a stand-alone SVG or an R igraph
